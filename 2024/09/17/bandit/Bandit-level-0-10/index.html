pwnable
bandit

Bandit Level 0

Level Goal

The goal of this level is for you to log into the game using SSH. The host to which you need to connect is bandit.labs.overthewire.org, on port 2220. The username is bandit0 and the password is bandit0. Once logged in, go to the Level 1 page to find out how to beat Level 1.

Analyze

처음 접속 시 디렉토리에서 readme 파일을 볼 수 있다.

readme 파일을 읽어 다음 password를 확인

Password

Bandit Level 1

Level Goal

The password for the next level is stored in a file called - located in the home directory

Analyze

접속하면 디렉토리에서 “-“ 파일을 확인할 수 있다.

cat 명령어를 사용하여 “-“ 파일을 읽으려 시도하니, -가 cat의 옵션입력으로 인식하여 명령이 제대로 실행되지 않는다.

./ 를 붙여 현재 디렉토리의 “-“ 파일을 읽어올 수 있도록 해 password를 확인하였다.

Password

Bandit Level 2

The password for the next level is stored in a file called spaces in this filename located in the home directory

Analyze

접속하면 디렉토리에서 “spaces in this filename”이라는 파일을 볼 수 있다.
일반적으로 cat 명령을 통해 해당 파일을 읽을 경우, 공백으로 인해 제대로 파일을 읽어올 수 없다.

“ 를 활용하여 cat 명령을 통해 공백이 존재하는 파일을 읽어올 수 있다.

Password

Bandit Level 3

The password for the next level is stored in a hidden file in the inhere directory.

Analyze

접속하면, 다음과 같이 inhere 디렉토리를 확인할 수 있다.

inhere 디렉토리 내에 다른 파일을 확인할 수 없는데,

ls -al 명령을 사용하여 숨겨진 파일까지 확인할 수 있다.

숨겨진 파일을 읽어 Password 확인

Password

Bandit Level 4

The password for the next level is stored in the only human-readable file in the inhere directory. Tip: if your terminal is messed up, try the “reset” command.

Analyze

접속 시 inhere 디렉토리를 확인할 수 있다.

들어가면 -file00 ~ -file09까지의 파일을 확인할 수 있는데,

-file07에서 원하는 Password를 확인할 수 있다.

Password

Bandit Level 5

The password for the next level is stored in a file somewhere under the inhere directory and has all of the following properties:

human-readable
1033 bytes in size
not executable

Analyze

접속하면 동일하게 inhere 디렉토리를 확인할 수 있다.

inhere 디렉토리 내에 수많은 다른 디렉토리를 확인할 수 있는데,

find 명령어로 1033 byte의 크기를 가진 파일을 찾을 수 있다.

cat 명령어로 .file2를 읽어 Password를 확인할 수 있다.

Password

Bandit Level 6

The password for the next level is stored somewhere on the server and has all of the following properties:

owned by user bandit7
owned by group bandit6
33 bytes in size

Analyze

접속 시 디렉토리에서 다른 특별한 파일이나 디렉토리를 확인할 수 없다.

find 명령어를 통해 bandit7 유저의 파일 중, size가 33 byte인 파일을 찾을 수 있다.

아래 dpkg 디렉토리에서 Password를 확인할 수 있다.

Password

Bandit Level 7

The password for the next level is stored in the file data.txt next to the word millionth

Analyze

접속 시 data.txt 파일을 확인할 수 있다.

파일 안에 상당히 많은 더미 데이터를 볼 수 있는데,

grep 명령어를 활용하여 millionth의 Password를 확인할 수 있다.

Password

Bandit Level 8

The password for the next level is stored in the file data.txt and is the only line of text that occurs only once

Analyze

접속하면 동일하게, data.txt 파일을 확인할 수 있다.

data.txt를 읽어보면 엄청나게 많은 더미 데이터를 볼 수 있다.
이중에서 한번만 적혀 있는게 Password인데,

sort 명령어를 통해 정렬을 해서 확인해 보면, 한번만 적혀 있는 Password를 확인할 수 있다.

Password

Bandit Level 9

The password for the next level is stored in the file data.txt in one of the few human-readable strings, preceded by several ‘=’ characters.

Analyze

접속하면 동일하게 data.txt 파일을 확인할 수 있다.

더미 데이터 안에서 읽을 수 있는 Password를 찾을 수 있다.

Password

Bandit Level 10

The password for the next level is stored in the file data.txt, which contains base64 encoded data

Analyze

접속하면 data.txt 파일을 확인할 수 있다.

읽어보면, base64로 인코딩된 데이터를 볼 수 있다.

base64 명령어를 사용하여 -d 옵션을 통해 디코딩해 Password를 확인할 수 있다.
