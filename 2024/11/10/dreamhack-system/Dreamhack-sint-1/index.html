Dreamhack - sint[1]

bintab1e
2024-11-10
pwnable
Dreamhack

Description

이 문제는 서버에서 작동하고 있는 서비스(sint)의 바이너리와 소스 코드가 주어집니다.
프로그램의 취약점을 찾고 익스플로잇해 get_shell 함수를 실행시키세요.
셸을 획득한 후, “flag” 파일을 읽어 워게임 사이트에 인증하면 점수를 획득할 수 있습니다.
플래그의 형식은 DH{…} 입니다.

Aanlyze

문제 소스코드는 다음과 같다.

보호기법은 NX bit만 걸려있다.

문제에서 size 검증 로직이 존재하나, size가 0일 경우는 조건문에 적용되지 않는다. size에 0이 들어갈 경우 read 함수의 세 번쨰 인자가 -1이 된다.

이때, read 함수의 세 번째 인자는 size_t로 32 비트인 경우 unsigned int, 64 비트인 경우 unsigned long이다.

따라서, unsigned 타입이므로 -1일 경우 형변환으로 인해 4,294,967,295로 변하여 버퍼 오버플로우 트리거가 가능하다.
