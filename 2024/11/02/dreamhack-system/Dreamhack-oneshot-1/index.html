Dreamhack

Description

이 문제는 작동하고 있는 서비스(oneshot)의 바이너리와 소스코드가 주어집니다.
프로그램의 취약점을 찾고 셸을 획득한 후, “flag” 파일을 읽으세요.
“flag” 파일의 내용을 워게임 사이트에 인증하면 점수를 획득할 수 있습니다.
플래그의 형식은 DH{…} 입니다.

Analyze

문제 코드는 다음과 같다.

문제에는 canary를 제외한 모든 보호기법이 걸려 있다.

main 함수의 msg 배열의 크기는 16 바이트이지만, read 함수에서 46만큼 받고 있어 버퍼오버플로우가 가능하다.

주의할 점은 check 변수가 중간에 있는데, 해당 check 변수가 0 이상일 경우, 프로그램을 강제 종료한다. 따라서 payload를 덮어줄 때 check 변수의 위치를 0으로 신경써서 덮어주어야 한다.

이때, 문제에서 stdout 의 메모리 상 위치를 알려주고 있으므로, 해당 위치를 통해 libc_base를 구할 수 있다.

oneshot gadget을 구하고, 해당 gadget으로 return address를 덮어 해결하였다.
